2. TECHNICAL ARCHITECTURE
--------------------------------------------------------------------------------
Frontend:
- Framework: React (v19.1.1)
- Build Tool: Create React App (react-scripts v5.0.1)
- State Management: React Hooks & Location State
- Animations: GSAP (v3.15+) & Framer Motion (v12.38+)
- Styling: Tailwind CSS (v3.4+) with PostCSS Autoprefixer
- Icons: Lucide React, React Icons

Backend Interface:
- Technology: FastAPI (Python 3.11-slim)
- Current Scope: Health monitoring and API placeholder
- Port: 8000

Infrastructure:
- Containerization: Docker (Multi-stage builds for Frontend and Backend)
- Orchestration: Kubernetes (k8s)
- Ingress: Traefik IngressRoute
- Reverse Proxy: Nginx (serving the React build)
